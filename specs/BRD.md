# Business Requirements Document (BRD): Logic Gates Simulator

## 1. Project Overview

**Project Name:** Logic Gates Simulator  
**Type:** Python Interactive Application  
**Core Functionality:** An interactive digital logic circuit simulator that supports all basic logic gates (AND, OR, NOT, NAND, NOR, XOR, XNOR), custom circuit building, and truth table generation.

**Target Users:** Students learning digital logic, educators, and developers working with circuit design.

---

## 2. Features

- All basic gates: AND, OR, NOT, NAND, NOR, XOR, XNOR
- Custom circuit builder
- Truth table generator
- Built-in Half Adder and Full Adder circuits
